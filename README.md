# Multi-Agent-Reinforcement-Learning---Pacman
Implementation of MARL algorithms (IQL, QMix, QPLEX) for the Pacman Capture the Flag challenge. Features advanced exploration (Count-Based with state abstraction), Rainbow extensions (Dueling, N-Step), and Reward Shaping. Optimized for coordination in sparse-reward environments, achieving a 97% win rate against heuristic agents.
